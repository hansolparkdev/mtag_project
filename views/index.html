■ 로그인 페이지

1) 초기 비밀번호 변경



- 아이디는 admin으로 고정
- 사용자 메뉴얼에 명시된 초기 비밀번호 입력
- 초기 비밀번호로 접속시 비밀번호 설정 팝업 출력



- 인증번호 입력 후 인증 버튼을 통해 인증
- 인증번호는 사용자 메뉴얼을 통해서 제공됨
- 인증 확인 후 비밀번호 변경 가능



- 비밀번호 변경 성공 메시지 출력

2) 로그인



- 아이디 : admin
- 비밀번호는 변경한 비밀번호 입력 후 로그인

3) 사용자 정보가 다를 경우 처리



- 사용자 정보를 5회 이상 다를 경우 사용자 정보 초기화



- 사용자 정보를 5회 이상 다를 경우 사용자 정보 초기화



- ID, Password 입력창 비활성화 및 로그인 잠김
- 비밀번호 변경을 통해 비밀번호 변경
- 비밀번호 변경 후 로그인 창 활성화



- 인증번호 입력 후 인증 버튼을 통해 인증
- 인증번호는 사용자 메뉴얼을 통해서 제공됨
- 인증 확인 후 비밀번호 변경 가능



- 비밀번호 변경 성공 메시지 출력




■ 메인페이지



- #Onboard monitoring을 클릭을 통해 Monitoring 페이지 접속
- #Tag Management 클릭을 통해 Tag 관리 페이지 접속
- #Beacon Management 클릭을 통해 Beacon 관리 페이지 접속



■ 모니터링 페이지

- Tag 데이터가 20초간 들어오지 않을 경우 Tag의 위치 표시는 사라짐
- 응급상황은 예외로 20초간 Tag 데이터가 들어오지 않아도 위치 표시
  (관리자가 종모양 버튼을 클릭하여 직접 꺼줘야 함)

1) 모니터링 페이지 첫 화면



2) 위치 내 태그 정보 확인

 

- 맵 내 마커 클릭 시 우측에 해당 위치 태그 정보 출력

3) 응급상황 처리



- 응급 상황 발생 시 좌 상단의 응급 버튼 점멸



- 응급 버튼 클릭 시 현재 응급 상황인 태그 정보 표시



- 응급 해제할 대상 행 클릭 후 확인 창을 통해 응급 상황 해제 가능

4) 층 이동



- 우 상단에 표시되는 층 버튼을 이용해 층 이동 가능


■ Tag 관리 페이지

1) Tag 관리 페이지 첫 화면



2) Tag 추가



- Tag 추가 버튼 클릭 시 팝업창 출력



- TagID, Name, Type 입력 후 추가 버튼 클릭



- 알림 창을 통해 성공 메시지 출력
- 확인 버튼 클릭시 화면 갱신을 통해 Tag 추가 확인 가능

3) Tag 수정


- 수정할 행의 수정 버튼 클릭 시 팝업창 출력
- 이름, Type 수정 가능
- 수정 버튼을 통해 수정 가능



- 알림 창을 통해 수정 성공 메시지 확인
- 확인 버튼 클릭 시 화면 갱신을 통해 Tag 수정 확인 가능

4) Tag 삭제



- 삭제할 행의 삭제 버튼 클릭 시 팝업창 출력
- 삭제버튼 클릭 시 삭제 가능



- 알림 창을 통해 삭제 성공 메시지 확인
- 확인 버튼 클릭 시 화면 갱신을 통해 Tag 삭제 확인 가능

5) Tag 상태 확인



- 상태를 확인할 Tag 행의 상태확인 버튼 클릭



- 해당 Tag의 정보 확인 가능

6) 검색 기능



- 우측 상단 검색창을 통해 ID, Name, Type으로 검색 가능



■ Beacon 관리 페이지

1) Beacon 관리 페이지 첫 화면



2) Beacon 추가



- 비콘 추가 버튼 클릭 시 팝업창 출력



- Beacon ID 입력
- 배치할 층 선택
- 추가 버튼 클릭 시 추가 가능



- 알림 창을 통해 추가 성공 메시지 확인
- 확인 버튼 클릭 시 화면 갱신을 통해 Beacon 추가 확인 가능

3) Beacon 수정



- 수정할 행의 수정 버튼 클릭 시 팝업창 출력



- location 입력
- installed 상태 선택
  - 비콘의 설치 상태를 표시
  - 해당 위치에 비콘 설치 이후 관리자가 직접 installed 설치로 변경
  - 미설치 상태일 경우 모니터링 페이지에서 미표시
- 수정버튼을 통해 수정 가능



- 알림 창을 통해 수정 성공 메시지 확인
- 확인 버튼 클릭 시 화면 갱신



- Beacon 수정 확인 가능

4) Beacon 위치 지정



- 맵 내 이동할 Beacon 드래그
- 드래그 도중 해당 영역을 벗어날 경우 원래 위치로 초기화
- 드래그 후 확인 창을 통해 적용 여부 확인
- 확인 시 해당 위치 저장



- 알림 창을 통해 위치 적용 성공 메시지 확인



- 드래그시 맵을 벗어나게 될 경우 경고창 출력

5) Beacon 삭제




- 삭제할 행의 삭제 버튼 클릭 시 팝업창 출력
- 삭제버튼 클릭 시 삭제 가능



- 확인 버튼 클릭 시 화면 갱신



- 맵과 테이블에서 마커 삭제 확인 가능



■ Tag 사용 방법

- 태그 LoRa 전송 주기는 10초 간격
- 응급 버튼을 눌렀을 경우 다음 LoRa 전송 때 응급신호를 보냄







■ Gateway 사용 방법



1) 게이트웨이 후면 검은원에 5V 파워 어댑터를 연결한다.
2) 파란원에는 이더넷 케이블을 연결해준다.
3) 빨간원의 스위치를 ON으로 맞춘다.



4) 전면부의 스위치(검은원)를 누르면 게이트웨이가 실행된다.
